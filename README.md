# eslint-config-free

## Get Started

### Install

```
yarn add -D eslint-config-free babel-eslint@10.x eslint@6.x
```

`babel-eslint` and `eslint` are required peer dependencies.

### Extend the config

Ensure the `free` config is in the `extends` in your `.eslintrc` file.

```json
{
  "extends": "free"
}
```

## Includes

- typescript linting
- prettier support
