# eslint-config-free

## Get Started

### Install

```
npm i -D eslint-config-free
```

### Extend the config

Ensure the `free` config is in the `extends` in your eslint config file.

```json
{
  "extends": "free"
}
```

## Includes

- typescript linting
- eslint-config-prettier
- eslint-plugin-import
- eslint-plugin-jsx-a11y
- eslint-plugin-react
- eslint-plugin-react-hooks
